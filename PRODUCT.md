# AutoWhisper landing site

## Register

Brand.

## Product purpose

AutoWhisper is a local-first desktop dictation app for people who want fast speech-to-text without sending audio to a cloud transcription service. The landing page exists to make the distribution path public while the main application repository remains private or transitions toward public release.

## Audience

- macOS users who want a notarized desktop app they can download and run without a developer setup.
- Linux users comfortable with terminal install commands and PPA package sources.
- Privacy-conscious engineers, writers, founders, and operators who dictate notes, code review comments, messages, and long-form drafts.
- Open-source evaluators checking whether the project is serious, maintained, and direct about distribution limitations.

## Positioning

AutoWhisper should feel like a precise desktop instrument, not a growth SaaS. It is calm, local, and operationally honest. The promise is simple: press a hotkey, speak, get text, keep audio on your machine.

## Tone

Plainspoken, specific, and evidence-led. Prefer concrete nouns over marketing abstractions. Avoid hype words, fake urgency, and generic launch-page language. The site should read like it was written by the maintainer of a serious native utility.

## Required content

- macOS download CTA for the v0.7.0 notarized build.
- Latest release link.
- Ubuntu and Debian PPA install commands.
- Local-first and privacy positioning.
- GitHub repository link.
- Honest caveat that the main app repository and release assets may still be private, so public downloads may require the repo or mirrored assets to become public.

## Anti-references

- Generic AI SaaS hero pages with gradient text, floating glass cards, vague productivity claims, and analytics scripts.
- Overbuilt JavaScript landing stacks for a static utility site.
- Dark neon developer-tool pages that look interchangeable.

## Design ambition

A tactile, editorial page that feels like a field manual for a quiet piece of desktop infrastructure. Static HTML and CSS only. No runtime JavaScript. No tracking.

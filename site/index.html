Linux install

Ubuntu / Debian via PPA

Add the AutoWhisper package source, refresh apt, then install the desktop utility. These commands are shown exactly so the path is easy to audit. 

sudo add-apt-repository ppa:primemanifold/autowhisper
sudo apt update
sudo apt install autowhisper

Privacy posture

A clear boundary: your speech should not need a remote transcription service.

AutoWhisper is positioned around local processing, local models, and a desktop workflow that does not require cloud speech APIs for normal dictation. That makes the behavior easier to reason about and easier to inspect.

The claim is intentionally narrow: local-first transcription, not magic security dust. You still control where the resulting text is pasted, saved, or sent.
